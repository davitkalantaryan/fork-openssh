UDOOCS





UDOOCS is a fork of the projet DOOCS . The Code is modified sligtly in order to compile it on WINDOWS (~20 one line modification , mainly added code for WINDOWS using #ifdef _WIN32 approach) . Up to now code compiled and tested for following combination of OS+architecture+compiler

1. WINDOWS 7,8,10 using microsoft compiler (VS 2013 and VS 2015), architecture ARM,AMD64,x86
2. LINUX for several distros (Ubuntu(12,14,16), SL(5,6,7))

Most probably the code will be possble to compile for any taget supportet by original DOOCS . In order to compile modified code on WINDOWS WLAC library is used. . WLAC stands for Windows Unix Api Converter. More details on this library will be available soon . WLAC itself does not introduce dependencies. Finall DOOCS libraries depend on wlac.lib and rfc.lib (rfc stands for Remote Function Call, and implements Unix like signalling for WINDOWS (kill, sigquee and friends))
